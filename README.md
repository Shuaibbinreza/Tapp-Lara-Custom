# TappLara v1

## Base Application - Laravel 11

### Requirement

-   PHP >= 8.2
-   Laravel 11

### Deployment Instruction

-   `git clone https://gitlab.com/tappware/tapp-lara.git`
-   `composer install`
-   Copy `.env.example` to `.env`
-   Update database credentials in `.env`
-   `php artisan migrate`
-   `php artisan serve`
